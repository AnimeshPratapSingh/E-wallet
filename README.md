
## Build a basic version of PayTM
E-wallet web Application
• Developed a E-wallet web application using MERN stack project
• Aweb application that allows users to securely transfer money to others, offering a seamless sign-in and sign-up process.
The application is designed with an intuitive user interface.
• Tools and technologies used: React.js, Tailwind, JavaScript, MongoDB, Express.js, Node.js
